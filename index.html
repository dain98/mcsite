$$Cost\:=\: \frac{1}{n} \displaystyle\sum _1^n\frac{N_i}{M_i}\:\cdot \sqrt[3]{\frac{n}{m}}$$

N = the player score for each map
M = the median score for each map
n = the number of maps played by a player
m = median of the total number of maps played
DISCLAIMER: Mutiple MP links ONLY work when they're from the same tournament AND same round (meaning all players are using the same pool)!
